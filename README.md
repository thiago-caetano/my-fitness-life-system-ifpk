# Sistema de Fitness

### Sobre o projeto

Este projeto foi desenvolvido durante a imersão **IA para Devs**, utilizando ferramentas modernas com apoio de Inteligência Artificial para acelerar o desenvolvimento da aplicação.

Durante a imersão foram utilizadas tecnologias e ferramentas como:

* Next.js
* React
* TypeScript
* Supabase
* Tailwind CSS
* Cursor
* v0
* Vercel

A proposta do projeto foi criar uma base funcional de um sistema fitness em um curto período de tempo, focando principalmente em produtividade, prototipação rápida e deploy.

### Importante

Este repositório representa a versão criada durante o dia da imersão, então ele será mantido como uma versão única/demonstrativa do projeto.

Uma nova versão, com mais funcionalidades, melhorias na arquitetura, autenticação aprimorada, novas telas e recursos adicionais, está sendo desenvolvida de forma privada.

Quando essa nova versão estiver pronta, ela será publicada futuramente no GitHub.

### Funcionalidades atuais

* Estrutura inicial do sistema
* Base para autenticação
* Integração com Supabase
* Organização de componentes reutilizáveis
* Interface moderna e responsiva
* Deploy utilizando Vercel

### Como executar o projeto

```bash
# Clonar o repositório
git clone <repo-url>

# Entrar na pasta
cd sistema-de-fitness

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```

O projeto ficará disponível em:

```bash
http://localhost:3000
```
